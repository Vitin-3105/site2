Estrutura e como navegar pelo site

  • body: contém tudo que está no visual, inclusive via botão de lista de comandos
  • header: É o cabeçalho do site. Fica o topo do site.
  • nav: O próprio nome já diz, navegação, usado para links que nos levam a outras partes do site.
  • main: Dentro do comando <main> você referencia todo o conteúdo principal (como esta lista)
  • aside: Itens que podem ser colocados “à parte” como links relacionados.
  • footer: Parte inferior do site, com informações adicionais.

Essa estrutura é clássica e já ajuda seu site a ser semântico!

Dicas de comandos

  • O comando de parágrafo começa com a tag <p>.
  • Na mesma proporção se deseja colocar título, coloque dentro de tags de título (<h1>, <h2>, etc).
  • Para destacar algo com negrito, use a tag <strong>. Para itálico, use a tag <em>.
  • Assin adicionar um botão com estilo é com o botão do vscode: botão direito > inserir snippet

Estilo e texto

  • Use a tag <style> dentro do <head> ou em arquivo separado (.css) para definir cor, fundo, tamanho de texto e fontes.
  • Ex: comando de título como <h1> { font-size: 24px; }.
